TiBubble Module

Description

TiBubble is a Titanium native module that provides a configurable bubble speech view for iOS and Android. It renders a speech bubble with customizable corner radius, fill color, beak position, and tail geometry.

The BubbleView can be used as a container for other views (labels, images, etc.) by adding them as children.

Platforms

| Platform | Minimum Version | Supported | |----------|----------------|-----------| | iOS | 11.0+ | ✅ | | Android | 5.0+ (API 21+) | ✅ |

Requirements

  • Titanium SDK: 13.2.0.GA or later
  • Architectures:
    • iOS: arm64, x86_64 (Simulator)
    • Android: armeabi-v7a, arm64-v8a, x86, x86_64

Installation

  1. Download the latest distribution package for your platform(s):
    • iOS: de.marcbender.bubbleview-iphone-1.0.0.zip
    • Android: de.marcbender.bubbleview-android-1.0.0.zip
  2. Place the .zip file(s) in your Titanium project's modules directory
  3. Add the module to your tiapp.xml:

xml <modules> <module platform="iphone" version="1.0.0">de.marcbender.bubbleview</module> <module platform="android" version="1.0.0">de.marcbender.bubbleview</module> </modules>

  1. Build and run your Titanium project

Accessing the Module

javascript const tibubble = require('de.marcbender.bubbleview');

API Reference

Module Constants

| Constant | Value | Description | |----------|-------|-------------| | tibubble.BUBBLE_BEAK_LEFT | 0 | Beak points left | | tibubble.BUBBLE_BEAK_RIGHT | 1 | Beak points right | | tibubble.BUBBLE_BEAK_LOWER | 0 (iOS: 2) | Beak points downward | | tibubble.BUBBLE_BEAK_UPPER | 1 (iOS: 3) | Beak points upward |

Note: On Android, bubbleBeak and bubbleBeakVertical both use 0/1. On iOS, the values are mapped internally (0/1 for horizontal, 2/3 for vertical). The JavaScript API is consistent across platforms.

Module Methods

tibubble.createBubble(properties)

Creates a new BubbleView instance.

Parameters:

| Name | Type | Description | |------|------|-------------| | properties | Object | Dictionary of properties to apply to the bubble |

Returns: BubbleProxy - The created bubble view proxy

Example:

javascript const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLE_BEAK_LEFT, bubbleBeakVertical: tibubble.BUBBLE_BEAK_LOWER, tailWidth: 4, tailLength: 10, tailCurveY: 4, width: 200, height: 150, top: 50, left: 50 });

BubbleView Properties

All properties can be set at creation time or modified at runtime.

bubbleColor (String)

The fill color of the bubble. Accepts any valid Titanium color value.

| Attribute | Value | |-----------|-------| | Type | String | | Default | #FFFFFF (white) | | Supported Formats | Hex (#RRGGBB), RGB, RGBA, Named colors ('red', 'blue', etc.) |

`javascript // Set at creation const bubble = tibubble.createBubble({ bubbleColor: '#4183C4' });

// Change at runtime bubble.bubbleColor = '#ff0000'; bubble.bubbleColor = 'red'; bubble.bubbleColor = Ti.Color.RED; `

bubbleRadius (Number)

The corner radius of the bubble in points/dp. Determines how rounded the bubble corners are.

| Attribute | Value | |-----------|-------| | Type | Number | | Default | 20.0 | | Unit | points (iOS) / dp (Android) |

`javascript // Set at creation const bubble = tibubble.createBubble({ bubbleRadius: 20 });

// Change at runtime bubble.bubbleRadius = 30; `

bubbleBeak (Number)

The horizontal position of the bubble's beak (pointer).

| Attribute | Value | |-----------|-------| | Type | Number | | Default | tibubble.BUBBLE_BEAK_LEFT (0) | | Valid Values | 0 (left), 1 (right) |

`javascript // Set at creation const bubble = tibubble.createBubble({ bubbleBeak: tibubble.BUBBLEBEAKLEFT });

// Change at runtime bubble.bubbleBeak = tibubble.BUBBLEBEAKRIGHT; `

bubbleBeakVertical (Number)

The vertical position of the bubble's beak (pointer).

| Attribute | Value | |-----------|-------| | Type | Number | | Default | tibubble.BUBBLE_BEAK_LOWER (0) | | Valid Values | 0 (lower/downward), 1 (upper/upward) |

`javascript // Set at creation const bubble = tibubble.createBubble({ bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER });

// Change at runtime bubble.bubbleBeakVertical = tibubble.BUBBLEBEAKUPPER; `

tailWidth (Number)

The width of the bubble's tail (beak) in points/dp. Controls how wide the tail is at its base.

| Attribute | Value | |-----------|-------| | Type | Number | | Default | 4.0 | | Unit | points (iOS) / dp (Android) |

javascript bubble.tailWidth = 6;

tailLength (Number)

The length of the bubble's tail (beak) in points/dp. Controls how far the tail extends from the bubble body.

| Attribute | Value | |-----------|-------| | Type | Number | | Default | 10.0 | | Unit | points (iOS) / dp (Android) |

javascript bubble.tailLength = 15;

tailCurveY (Number)

The curve offset of the bubble's tail in points/dp. Controls how curved the tail is.

| Attribute | Value | |-----------|-------| | Type | Number | | Default | 4.0 | | Unit | points (iOS) / dp (Android) |

javascript bubble.tailCurveY = 6;

Standard Titanium View Properties

In addition to the bubble-specific properties, the BubbleView supports all standard Titanium view properties:

| Property | Type | Description | |----------|------|-------------| | width | Number/String | Width of the view | | height | Number/String | Height of the view | | top | Number/String | Top position | | left | Number/String | Left position | | right | Number/String | Right position | | bottom | Number/String | Bottom position | | zIndex | Number | Stack order | | opacity | Number | Opacity (0.0 - 1.0) | | visible | Boolean | Visibility | | touchEnabled | Boolean | Touch handling | | layout | String | Layout mode ('vertical', 'horizontal', 'none') |

BubbleView Methods

All standard Titanium view methods are available (add(), remove(), animate(), etc.).

Events

The BubbleView supports all standard Titanium view events:

| Event | Description | Properties | |-------|-------------|------------| | click | Fired when the bubble is tapped | source, x, y | | singletap | Fired on single tap | source, x, y | | doubletap | Fired on double tap | source, x, y | | longpress | Fired on long press | source, x, y | | touchstart | Fired when touch begins | source, x, y | | touchend | Fired when touch ends | source, x, y | | touchmove | Fired when touch moves | source, x, y |

javascript bubble.addEventListener('click', function(e) { Ti.API.info('Bubble tapped at: ' + e.x + ', ' + e.y); });

Examples

Basic Bubble

`javascript const tibubble = require('de.marcbender.bubbleview');

const win = Ti.UI.createWindow({ backgroundColor: '#f0f0f0' });

const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLEBEAKLEFT, bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER, width: 200, height: 150, top: 50, left: 50 });

win.add(bubble); win.open(); `

Bubble with Text Content

`javascript const tibubble = require('de.marcbender.bubbleview');

const win = Ti.UI.createWindow({ backgroundColor: '#f0f0f0' });

const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLEBEAKLEFT, bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER, width: 200, height: 150, top: 50, left: 50 });

const label = Ti.UI.createLabel({ text: 'Hello, World!', color: '#ffffff', font: { fontSize: 18, fontWeight: 'bold' }, height: Ti.SIZEAUTO, width: Ti.SIZEAUTO, top: 20, left: 20, right: 20 });

bubble.add(label); win.add(bubble); win.open(); `

Interactive Bubble with Click Handler

`javascript const tibubble = require('de.marcbender.bubbleview');

const win = Ti.UI.createWindow({ backgroundColor: '#f0f0f0' });

const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLEBEAKRIGHT, bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER, width: 200, height: 150, top: 50, left: 150 });

const label = Ti.UI.createLabel({ text: 'Tap me!', color: '#ffffff', font: { fontSize: 18, fontWeight: 'bold' }, height: Ti.SIZEAUTO, width: Ti.SIZEAUTO, top: 20, left: 20 });

bubble.add(label); win.add(bubble); win.open(); `

Animated Bubble

`javascript const tibubble = require('de.marcbender.bubbleview');

const win = Ti.UI.createWindow({ backgroundColor: '#f0f0f0' });

const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLEBEAKLEFT, bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER, width: 200, height: 150, top: 50, left: 50 });

bubble.addEventListener('click', function() { bubble.animate({ duration: 500, left: bubble.left === 50 ? 200 : 50, top: bubble.top === 50 ? 200 : 50, callback: function() { Ti.API.info('Animation complete'); } }); });

Complete Interactive Demo

`javascript const tibubble = require('de.marcbender.bubbleview');

const win = Ti.UI.createWindow({ backgroundColor: '#f0f0f0' });

// Create bubble const bubble = tibubble.createBubble({ bubbleColor: '#4183C4', bubbleRadius: 20, bubbleBeak: tibubble.BUBBLEBEAKLEFT, bubbleBeakVertical: tibubble.BUBBLEBEAKLOWER, width: 200, height: 150, top: 50, left: 50 });

const label = Ti.UI.createLabel({ text: 'Bubble: Left-Lower', color: '#333', font: { fontSize: 16, fontWeight: 'bold' }, top: 220, left: 50 });

// Button: Beak Left const buttonLeft = Ti.UI.createButton({ title: 'Beak Left', top: 260, left: 50, width: 100 }); buttonLeft.addEventListener('click', function() { bubble.bubbleBeak = tibubble.BUBBLEBEAKLEFT; label.text = 'Bubble: Left-' + (bubble.bubbleBeakVertical === 0 ? 'Lower' : 'Upper'); });

// Button: Beak Right const buttonRight = Ti.UI.createButton({ title: 'Beak Right', top: 260, left: 160, width: 100 }); buttonRight.addEventListener('click', function() { bubble.bubbleBeak = tibubble.BUBBLEBEAKRIGHT; label.text = 'Bubble: Right-' + (bubble.bubbleBeakVertical === 0 ? 'Lower' : 'Upper'); });

// Button: Beak Upper const buttonUpper = Ti.UI.createButton({ title: 'Beak Upper', top: 310, left: 50, width: 100 }); buttonUpper.addEventListener('click', function() { bubble.bubbleBeakVertical = tibubble.BUBBLEBEAKUPPER; label.text = 'Bubble: ' + (bubble.bubbleBeak === 0 ? 'Left' : 'Right') + '-Upper'; });

// Button: Random Color const buttonColor = Ti.UI.createButton({ title: 'Random Color', top: 310, left: 160, width: 100 }); buttonColor.addEventListener('click', function() { const randomColor = '#' + Math.floor(Math.random() * 16777215).toString(16).padStart(6, '0'); bubble.bubbleColor = randomColor; label.text = 'Color: ' + randomColor; });

// Button: Radius +10 const buttonRadius = Ti.UI.createButton({ title: 'Radius +10', top: 360, left: 50, width: 100 }); buttonRadius.addEventListener('click', function() { bubble.bubbleRadius = (parseFloat(bubble.bubbleRadius) || 20) + 10; label.text = 'Radius: ' + bubble.bubbleRadius; });

win.add(bubble); win.add(label); win.add(buttonLeft); win.add(buttonRight); win.add(buttonUpper); win.add(buttonColor); win.add(buttonRadius); win.open(); `

Platform Differences

iOS

  • Uses Core Graphics for rendering
  • Supports hardware acceleration
  • Beak values: 0/1 for horizontal, 2/3 for vertical (mapped internally)
  • Fallback to simple rectangle if view is too small

Android

  • Uses Canvas/Path for rendering
  • Anti-aliasing enabled by default
  • Beak values: 0/1 for both horizontal and vertical
  • Fallback to rounded rectangle if view is too small

Tips & Best Practices

  1. Minimum Size: Ensure the bubble is at least 2 * bubbleRadius in width and height for proper rendering
  2. Performance: The bubble redraws on every property change. Batch property changes where possible
  3. Content: Add child views with appropriate margins to avoid overlap with the bubble edges
  4. Tail Customization: Adjust tailWidth, tailLength, and tailCurveY together for consistent proportions
  5. Memory: The bubble uses minimal memory. No special cleanup is required
